# O'zbek Tilini Matematik Modellashtirish

Bu dastur O'zbek tilini matematik modellashtirish va amaliy tilshunoslik mavzusidagi ilmiy-amaliy taqdimotni namoyish etish uchun yaratilgan interaktiv Flutter ilovasi.
Dastur veb, desktop va mobil qurilmalarda ishlashga to'liq moslashtirilgan.

## Dastur imkoniyatlari
- Interaktiv slaydlar ko'rinishidagi ma'lumotlar
- Har bir slaydda 8 tadan ilmiy va amaliy faktlar
- O'zbek tilshunoslari va ularning ishlari haqida batafsil ma'lumotlar
- Mobil qurilmalar va kompyuterlar uchun moslashuvchan (responsive) interfeys
- Yon panel orqali slaydlarga oson o'tish imkoniyati (desktop rejimida)

## Texnologiyalar
- Flutter
- Dart
- Material 3 dizayn

## Ishga tushirish
Dasturni ishga tushirish uchun quyidagi buyruqdan foydalaning:
```bash
flutter run
```
